&nbsp;What is the SmartKey Starter®?

  • The SmartKey Starter is a seamless remote start solution for Mercedes-Benz passenger vehicles and Sprinter vans.

How does it work?

  • Out of the box, the SmartKey Starter® will work off of the factory key fob. &nbsp;All passenger (non-Sprinter) vehicles can be remotely started by hitting the ‘panic’ button one time (if the OEM key is equipped with a panic button). &nbsp;The panic feature will work as designed by holding down the panic button. &nbsp;If the vehicle does not have a panic button on the OEM key, you will be able to use a ‘lock-unlock-lock’ sequence to start the vehicle remotely. The 2007-2018 Sprinter vans will remotely start by hitting the ‘lock’ button on the factory remote three times. &nbsp;The 2019+ Sprinters (SKSNG907 and SKSNG907RV will remotely start using a lock-unlock-lock sequence from the OEM key fob. There are also several ways to add extended-range remote starting, lock, and unlocking capabilities. &nbsp;The&nbsp;Compustar Drone&nbsp;or&nbsp;Directed&nbsp;Smart Start&nbsp;can be added to the SmartKey Starter® and will allow for operation from a smartphone or computer anywhere in the world. &nbsp;Extended-range RF remote kits from either Compustar or Directed will also work with the SmartKey Starter®. &nbsp;&nbsp;

Adding smartphone integration:&nbsp;will give you unlimited range. &nbsp;As long as the module in your vehicle has a cell signal and you can get on the internet, you will be able to remote start, lock, unlock, and release the trunk (if applicable). &nbsp;This is the most popular way to extend the range. &nbsp;This feature is an additional module that can be plugged into the SmartKey Starter®. &nbsp;The&nbsp;Compustar Drone&nbsp;is available for purchase at a discounted rate as a package with the SmartKey Starter®. &nbsp;When adding to your cart, check the box for smartphone control. &nbsp; There is a&nbsp;yearly fee&nbsp;for the Compustar Drone. &nbsp;The&nbsp;Directed Smart Start&nbsp;is also compatible with the SmartKey Starter® and also gives you the same features. &nbsp;There is a yearly fee for the Smart Start as well. &nbsp;Both the Drone and Smart Start offer premium plans that allow for GPS services such as vehicle tracking, speed notifications, geo-fencing, and more. &nbsp;All SKUs with ‘SKSNG’ in the part # will work with both Compustar and Directed. &nbsp;For all other models, orders for Directed compatible units will need to be placed over the phone at 312-421-1114.

Adding RF remote kits:&nbsp;RF remote kits are external fobs that can be used to operate the SmartKey Starter®. &nbsp;Every Compustar remote kit&nbsp;will work with the SmartKey Starter®. &nbsp;Adding&nbsp;Directed remotes&nbsp;requires an&nbsp;XL202&nbsp;RF to the data transmitter and any remote and antenna combination that works with the XL202. &nbsp;Both manufacturers offer one-way or two-way remote options. One-way remotes transmit a signal to the vehicle and two-way remotes transmit information both to the vehicle and back to the remote.

How long will it run?

  • For most passenger vehicles (vehicles other than Sprinters), the remote start factory setting will run for 15 minutes.
  • For the Sprinter van (SKS906C, SKS906D, SKSNG907, SKSNG907RV), the stand-alone remote start time is 1-hour. For SKS906CRV, SKS906-SMRT, and SKS906-SMRT-B, the default run time is 2 hours. If the remote start command is initiated through an external Compustar or Directed alarm / remote start module (for example, using a Compustar Drone or RF remote connected to a Compustar module), the run time will be dictated by the device that the remote start command was initiated through (for example, Compustar remote start brains can be programmed for 15-45 minute run-times). If initiating the remote start command through the OEM key fob or through a device connected directly to the SmartKey Starter® module, the run time will be the default setting (2-hour default for this system). If a shorter run time is desired, please contact us for custom programming.&nbsp;For all Next Generation (with SKSNG in the part #) hardware units, the run-time is field programmable.&nbsp; For all other SKUs, custom run time needs to be programmed upon ordering.&nbsp;&nbsp;
  • Note: when using a Sprinter auto start (SKSNG907, SKSNG907RV, SKS906CRV, SKS906-SMRT, SKS906SMRT-B) with external 1-wire trigger (latched ground to start and run) the system will re-start if the ground is still applied to start wire after run-time expiration. The system will continue running and re-starting until the external trigger removes the ground signal.&nbsp;

How is it installed?

  • The SmartKey Starter® is a revolutionary way to install a remote starter. &nbsp;Nearly all models are 100% plug-and-play and none of our units require any programming. &nbsp;Once you install the system and run the engine for 30 seconds, remote start will work right away. &nbsp;All SmartKey Starter® kits come with a T-harness that plugs in behind the Electronic Ignition Cylinder. &nbsp;Our T-harnessing requires no cutting or soldering. &nbsp;All of our harnessing comes with OEM connectors to ensure reliability and ease in installation. &nbsp;Most vehicles with Keyless Go (push-button start) require a jumper harness to either a CAN distribution block or a Keyless Go module. &nbsp;This harness also requires no cutting of wires or soldering.

Who can install the SmartKey Starter®?

  • Professional installation is always recommended. &nbsp;You can locate an&nbsp;authorized dealer and installer near you,&nbsp;or&nbsp;purchase the system online&nbsp;and bring to a shop that you feel comfortable with. &nbsp;There have been many end-users who have installed the SmartKey Starter® themselves. &nbsp;You can view all of our&nbsp;installation and disassembly manuals&nbsp;to see if the project is something you or you and your installer are comfortable with. &nbsp;We are available Monday-Friday for&nbsp;technical support.

Does the installation require any specialty tools?

  • In most cases, the tools required for the SmartKey Starter® installation are standard torx screwdrivers and panel removal tools. &nbsp;However, for some models, removing the EIS (electronic ignition cylinder) is either required or very helpful for the installation. &nbsp;To remove the EIS, some models require using an EIS removal tool to remove the metal ring that holds the EIS in the vehicle’s dash. &nbsp;This tool is&nbsp;not&nbsp;required for all models. You can see the full&nbsp;list of required or recommended vehicles here&nbsp;and&nbsp;purchase the EIS removal tool here.&nbsp; The tool is $29 and is an OEM Mercedes tool.

How will my climate control settings work?

  • Upon remote start, the climate control settings will remain the same as they were the last time the vehicle was shut off. &nbsp;To ensure optimal comfort, set the climate control to the desired temperature before you shut the vehicle off.

Will my heated seats turn on automatically?

  • For all Next Generation (NG) hardware units&nbsp;(with ‘SKSNG’ in the part #&nbsp;), the heated seats will turn on when the outside temperature is below a programmable threshold. &nbsp;The default setting is &lt; 41* F. &nbsp;From the steering wheel, you can set the temperature threshold to &lt;32* F or &lt;50* F or return to the factory setting. &nbsp;Any temperature can be set from a computer using a Compustar&nbsp;programmer and computer.&nbsp;

Will my rear defroster come on?

  • For all Next Generation (NG) hardware units (with ‘SKSNG’ in the part #&nbsp;), the rear defroster will turn on when the outside temperature is below the same programmed threshold as the heated seats.

What about my cooling seats?

  • For all Next Generation (NG) hardware units with the exception of the W166 platform&nbsp;(&nbsp;SKSNG166D3,&nbsp;SKSNG166D4&nbsp;) cooling seats can be programmed to come on at a desired temperature using a Compustar programmer and a computer. &nbsp;If you contact us, we can pre-program the unit to trigger the cooling seats at a desired temperature if your model is applicable. &nbsp;

What is the range of the factory key fob?

  • For passenger vehicles, Mercedes lists the factory key fob range at 200 feet. &nbsp;The Sprinter is about 30 feet. The Smartkey Starter® will be effective from a key fob at the same maximum distance that you can currently lock and unlock the doors using the factory remote. &nbsp;If you find that your range is far below the Mercedes listed range, you may need to replace the battery in your remote.

What if my vehicle does not have a ‘panic’ button?

  • Vehicles without a panic button that are covered by the SmartKey Starter® can still be remotely started using the factory remote. &nbsp;For passenger vehicles, there is a ‘lock-unlock-lock’ sequence that can be utilized for remote start. &nbsp;The 2007-2018 Sprinter Vans can be remotely started by hitting the lock button three times. 2019 and up Sprinters use a lock-unlock-lock sequence to remote start from the OEM key. &nbsp;&nbsp;&nbsp;

How do I program ‘lock-unlock lock’ remote start sequence?

  • For all Next Generation (NG) hardware units&nbsp;(all kits with ‘SKSNG’ in the part #), if no panic button is detected, the ‘lock-unlock-lock’ remote start sequence will be set automatically. &nbsp;For all other passenger vehicles, please consult this&nbsp;application note&nbsp;for the ‘lock-unlock-lock’ programming procedure. &nbsp; Sprinter vans (SKS906) do not require programming to utilize lock-lock-lock remote start. &nbsp;Once the system is installed and the valet switch is on, start the vehicle manually and let run for 30 seconds- remove the key, close all doors, and attempt to remote start. Please note: SKSNG907 and SKSNG907RV (2019+ Sprinter) use a ‘lock-unlock-lock sequence to remote start from the OEM key.
